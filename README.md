# KindleHandianSC

## 特点
只保留Kindle支持的词条
（不支持的反正也查不到，除非有在Kindle把字典当书读的需求）

保留汉典词库丰富的优点

删除了图片等在kindle上相对无用的数据

文件非常小

## 版本
KindleHanDianSC-1.mobi 只有“基本解释”

KindleHanDianSC-2.mobi 包含“基本解释”和“详细解释”

KindleHanDianSC-3.mobi 包含“基本解释”，“详细解释”，和“康熙字典”
